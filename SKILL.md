# 🍅 Tomato Novel Writer (番茄小说作家)

Fully automated novel writing and publishing for Tomato Novel platform.

## 🎯 Core Features

1. **Auto Writing** - Generate novel chapters based on genre/market demand
2. **Auto Publishing** - Upload chapters to Tomato Novel automatically (browser automation)
3. **Market Analysis** - Analyze reader data, retention rates, trends
4. **Auto Optimization** - Improve content based on performance
5. **Cover Generation** - Create book covers automatically
6. **Contract Management** - Track and apply for contracts
7. **Revenue Tracking** - Monitor earnings and royalties
8. **Full Auto Workflow** - End-to-end automation: write → publish → analyze → optimize
9. **Daily Auto Task** - Scheduled daily chapter releases

## 📖 Quick Start

Tell the agent:
- "写一本玄幻小说"
- "分析市场现在什么类型火"
- "发布新章节"
- "查看续读率数据"
- "优化第一章内容"

## 📊 Market Analysis Features

### Reader Metrics
- **续读率 (Retention Rate)** - Track chapter-to-chapter retention
- **完读率 (Completion Rate)** - How many finish the book
- **收藏数 (Favorites)** - Reader favorites count
- **评论数 (Comments)** - Engagement metrics
- **收入 (Revenue)** - Daily/monthly earnings

### Market Trends
- Hot genres this week
- Rising keywords
- Competitor analysis
- Reader demographics

## 🤖 Auto-Improvement Loop

```
1. Write chapter → 2. Publish → 3. Monitor data → 4. Analyze retention
                    ↓
6. Update strategy ← 5. Identify weak points
```

## 🔧 Configuration

- **Account:** Tomato Novel author account (login required)
- **Browser:** 360 Browser or Edge (for automation)
- **API:** Tomato Novel API (if available)
- **Storage:** `~/.openclaw/tomato-novel/` for drafts and data

## ⚠️ Requirements

1. **Tomato Novel Account** - Must have author account
2. **Login Session** - Browser must be logged in
3. **Content Guidelines** - Follow platform rules
4. **Original Content** - No plagiarism
